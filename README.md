# UI_RENDER

## 📌 프로젝트 소개

**UI_RENDER**는 다양한 화면에서 공통으로 사용할 수 있는  
재사용 가능한 UI 컴포넌트를 동적으로 생성하고 렌더링하기 위한 라이브러리입니다.

여러 서비스에서 일관된 UI 구조를 유지하고,  
공통 렌더링 로직을 표준화하는 것을 목표로 합니다.

---

## 🎯 목적

- 공통 UI 렌더링 로직 표준화
- 대시보드 컴포넌트 재사용성 향상
- 동적 UI 생성 지원

---
